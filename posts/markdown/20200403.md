Nuxt.jsのMiddlewareを使うことで、レンダリングするよりも前に実行される関数を定義できます。

## 実装してみる
実用性は皆無ですが、試しでMiddlewareの機能を使ってみます。

middlewareディレクトリにtest.jsを作成する。
```middleware/test.js
export default function() {
  console.log("test")
}
```

```nuxt.config.js
router: {
  middleware: "test"
}
```

```pages/index.vue
export default {
  middleware: 'test',
}
```

これで全てのページでレンダリングするよりも前にconsole.logにtestという文字が出ます。
