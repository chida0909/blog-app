# getters

- gettersはStoreの状態を算出する時に使います  
- プロパティスタイルアクセスは「this.$store.getters.hoge」と書く事ができます  
- ゲッターは「this.$store.getters.hoge」オブジェクトから取り出され、プロパティとしてアクセスできます  
- プロパティとしてアクセスされるゲッターは、キャッシュされます  

## gettersでageが30以上だけ表示

```js
import Vue from "vue"
import Vuex from "vuex"

Vue.use(Vuex)

export default new Vuex.Store({
  state: {
    users: [{ name: "John", age: 22 }, { name: "Merry", age: 33 }]
  },
  getters: {
    users(state) {
      return state.users.filter(user => user.age < 30)
    }
  }
})
```

```vue
<template>
  <ul>
    <li v-for="user in users">{{ user.name }} ({{ user.age }})</li>
  </ul>
</template>

<script>
export default {
  computed: {
    users() {
      return this.$store.getters.users
    }
  }
}
</script>
```

# mutations
- ミューテーションとは、ストアを更新するメソッドです  
- ミューテーションはタイプとハンドラを持ちます  
- ハンドラ関数はstateを第1引数として状態の変更を行います  
- ミューテーションハンドラを起動するには「this.$store.commit("hoge")」を呼び出す必要があります  

## stateをmutationsで更新

```js
import Vue from "vue"
import Vuex from "vuex"

Vue.use(Vuex)

export default new Vuex.Store({
  state: {
    count: 0
  },
  mutations: {
    increment(state) {
      state.count = 2
    }
  }
})
```

```vue
<template>
  <div>
    <h1>{{ count }}</h1>
  </div>
</template>

<script>
export default {
  data() {
    return {}
  },
  computed: {
    count() {
      this.$store.commit("increment")
      return this.$store.state.count
    }
  }
}
</script>
```

## stateをmutationsで更新(2)
クリックでカウントアップします

```js
import Vue from "vue"
import Vuex from "vuex"

Vue.use(Vuex)

export default new Vuex.Store({
  // stateはVue.jsでいうところのdataです
  state: {
    count: 0
  },
  mutations: {
    // カウントアップするミューテーションを登録
    increment(state) {
      state.count++
    }
  }
})
```

```vue
<template>
  <div>
    <button @click="increment">UP</button>
    <h1>Count:{{ count }}</h1>
  </div>
</template>

<script>
export default {
  methods: {
    // クリックイベントに合わせて、カウントアップするミューテーションを呼び出します
    increment() {
      this.$store.commit("increment")
    }
  },
  computed: {
    count() {
      // ストアの状態（count）を表示します
      return this.$store.state.count
    }
  }
}
</script>
```

# Actions

- アクションとはStoreを更新するメソッドです  
- ミューテーションと似ていますが、下記の点で異なります  
- アクションは状態を変更するのではなく、ミューテーションをコミットします  
- アクションは任意の非同期処理を含むことができます  
- context.commit('hoge')を呼び出すことでミューテーションをコミットできます。  
- アクションは「this.$store.dispatch('hoge')」がトリガーとなって実行されます。  
  
全体の処理の流れとしては、コンポーネントからdispatchでアクションを呼び出し、アクション内で外部APIなどからの非同期処理を行った後、commitでミューテーションを使いステートを更新するという流れとなります。  
  
## Actionsでstateを更新
1ずつカウントアップの処理で、Actionsを使いstateを更新します。
  

```js
import Vue from "vue"
import Vuex from "vuex"

Vue.use(Vuex)

export default new Vuex.Store({
  state: {
    count: 0
  },
  mutations: {
    increment(state) {
      state.count++
    }
  },
  actions: {
    incrementOne(context) {
      context.commit("increment")
    }
  }
})
```


```vue
<template>
  <div>
    <button @click="increment">UP</button>
    <h1>Count:{{ count }}</h1>
  </div>
</template>

<script>
export default {
  methods: {
    increment() {
      this.$store.dispatch("incrementOne")
    }
  },
  computed: {
    count() {
      return this.$store.state.count
    }
  }
}
</script>
```

## axiosで非同期通信
無料サービスJSONPlaceholderを利用します
  
ブラウザ上の画面
```
ユーザ一覧
Leanne Graham
Ervin Howell
Clementine Bauch
Patricia Lebsack
Chelsey Dietrich
```
  
```js
import Vue from "vue"
import Vuex from "vuex"
import axios from "axios"

Vue.use(Vuex)

export default new Vuex.Store({
  state: {
    users: []
  },
  mutations: {
    setUsers(state, users) {
      state.users = users
    }
  },
  actions: {
    getUsers({ commit }) {
      return axios
        .get("https://jsonplaceholder.typicode.com/users")
        .then(response => {
          commit("setUsers", response.data)
        })
    }
  }
})
```

```vue
<template>
  <div id="app">
    <h1>ユーザ一覧</h1>
    <div v-for="user in users" :key="user.id">{{ user.name }}</div>
  </div>
</template>

<script>
export default {
  name: "app",
  computed: {
    users() {
      return this.$store.state.users
    }
  },
  mounted() {
    // elementへのマウントが行われた後処理されます
    this.$store.dispatch("getUsers")
  }
}
</script>
```

# mapState

```vue
<template>
  <div>
    <h1>{{ count }}</h1>
  </div>
</template>

<script>
import { mapState } from 'vuex'

export default {
  data() {
    return {}
  },
  computed: {
    ...mapState({
      count: state => state.count
    }),
  }
}
</script>
```
