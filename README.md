# 開発環境の立ち上げ  
$ npm run dev

# MarkdownからJSONの変換  
$ npm run md

# ブログ記事（Markdown）を書く時の注意点
1.パーマリンクは重複させないこと（パーマリンクで一意の値を判断しているため）
2.crated_atは必須（指定しないと作成日が表示されない）
3.見出しはh1, h2を使用すること
